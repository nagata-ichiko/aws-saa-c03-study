# Q229 - RDS DB インスタンスの AZ 選択ページ

- **日付**: 2026-03-19
- **分野**: General
- **自分の回答**: B（DB INSTANCE DETAILS）
- **正解**: D（ADDITIONAL CONFIGURATION）

## 問題

RDS DB インスタンスを起動する際、どのページでアベイラビリティゾーンを選択できますか？

## 解説

アベイラビリティゾーンの選択は「Additional Configuration（追加設定）」ページで行う。DB INSTANCE DETAILS ではインスタンスクラスやエンジンバージョンなどの基本設定を行う。AZ はネットワーク関連の詳細設定に分類される。
