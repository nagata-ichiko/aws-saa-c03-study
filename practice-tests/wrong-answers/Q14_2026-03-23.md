## 問題

Amazon EC2 がサポートするディスクストレージの種類はどれですか？

## 解説

Amazon EC2 がサポートするディスクストレージには「Amazon Instance Store」（一時的なローカルストレージ）と「EBS (Elastic Block Store)」（永続的なブロックストレージ）があります。Instance Store はインスタンスに物理的に接続された一時ストレージで、インスタンスの停止・終了時にデータが消えます。AppStream や SNS はストレージサービスではありません。
