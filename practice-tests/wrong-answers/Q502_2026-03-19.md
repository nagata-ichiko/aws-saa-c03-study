# Q502 - EC2 Linux の /dev/sda1 の用途

- **日付**: 2026-03-19
- **分野**: Domain 4（コスト最適化アーキテクチャ）
- **自分の回答**: A（EBS ボリューム用に予約されている）
- **正解**: D（ルートデバイス用に予約されている）

## 問題

Amazon EC2 の Linux インスタンスで、デバイス名 /dev/sda1 はどのような用途に使われるか？

## 解説

/dev/sda1 はルートデバイス（起動ボリューム）用に予約されている。追加の EBS ボリュームには /dev/sd[f-p] が推奨される。ルートデバイス自体は EBS またはインスタンスストアのどちらかだが、デバイス名としてはルートデバイス専用。
