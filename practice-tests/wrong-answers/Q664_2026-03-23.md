- **正解**: B (Amazon S3 Storage Lens)

## 問題

グローバル企業が AWS 上でワークロードを運用しています。複数の AWS リージョンにまたがる Amazon S3 バケットを使用しており、バージョニングが有効になっていないすべての S3 バケットを特定したい場合、どのソリューションが要件を満たしますか？

## 解説

Amazon S3 Storage Lens は複数リージョンにまたがる S3 バケットの使用状況や設定を包括的に可視化でき、バージョニングの有無も確認可能です。AWS Config のルールはリージョン単位でのみ管理可能で、複数リージョン横断の可視化に不向きです。IAM Access Analyzer はアクセス許可解析が目的であり、S3 マルチリージョンアクセスポイントはアクセス簡略化のための機能です。
