# 間違えた問題: Q686

- **日付**: 2026-03-23
- **ドメイン**: Domain 4: コスト最適化アーキテクチャ
- **自分の回答**: C (S3 Standard-IA)
- **正解**: B (S3 Intelligent-Tiering)

## 問題

メディアファイルはAZの障害に耐えられる必要があり、アクセスパターンが予測できない（頻繁なものもあれば稀なものもある）場合に、コストを最小限に抑えるストレージオプションはどれか。

## 解説

S3 Intelligent-Tiering はアクセスパターンが予測できないファイルに対し、自動で最適なストレージクラス（高頻度・低頻度層）を選択しコストを最小化します。S3 Standard-IA は低頻度アクセス専用でアクセスコストが高く、頻繁にアクセスされる場合にコストが増大します。S3 One Zone-IA は単一 AZ のため AZ 障害への耐性が不足します。
