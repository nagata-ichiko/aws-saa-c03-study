# Q564 - RDS マルチ AZ 配置の読み書き担当

- **日付**: 2026-03-19
- **分野**: Domain 2（高可用性アーキテクチャ）
- **自分の回答**: A（セカンダリ）
- **正解**: D（プライマリ）

## 問題

DB インスタンスをマルチ AZ 配置で実行する場合、[　　] がデータベースの書き込みと読み取りを担当します。

## 解説

マルチ AZ 配置ではプライマリ DB インスタンスがすべての読み書きを担当する。スタンバイは別 AZ に同期レプリケーションされるが、トラフィックは受け付けない。障害時に自動フェイルオーバーする。リードレプリカとは別の概念。
