# 間違えた問題: S3 暗号化キー管理

- **日付**: 2026-03-22
- **モード**: ミニ模擬試験
- **問題番号**: Q7（セッション内）

## 問題文

ある企業が、Amazon S3 に保存されている機密データを保護する必要があります。要件：暗号化キーを自社で管理したい、キーのローテーションを自動化したい、キーの使用状況を監査したい。

## 選択肢

- A. SSE-S3（Amazon S3 マネージドキー）を使用する
- B. SSE-KMS（AWS KMS カスタマーマネージドキー）を使用する
- C. SSE-C（カスタマー提供キー）を使用する
- D. クライアントサイド暗号化を使用し、キーをオンプレミスの HSM で管理する

## 自分の回答: A（SSE-S3）

## 正解: B（SSE-KMS カスタマーマネージドキー）

## 解説

- SSE-S3: AWS が完全管理。自社でのキー管理・ローテーション制御・監査は不可
- SSE-KMS (CMK): 自社でキーポリシー管理 + 自動ローテーション + CloudTrail で監査可能
- SSE-C: 自社でキー提供するが、AWS 側で管理しないため自動ローテーション不可
- 「自社管理 + 自動ローテーション + 監査」の3要件 = SSE-KMS CMK が唯一の正解
