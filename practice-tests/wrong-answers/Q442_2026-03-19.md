# Q442 - 複数AZに展開すべきサービス

- **日付**: 2026-03-19
- **分野**: Domain 4（コスト最適化アーキテクチャ）
- **自分の回答**: A（DynamoDB）, B（EC2）
- **正解**: B（EC2）, C（Elastic Load Balancing）

## 問題

高可用性を実現するために複数のアベイラビリティゾーンに展開すべきサービスはどれか？（2つ選択）

## 解説

フルマネージドサービス（DynamoDB, S3, SNS など）はAWSが自動的に高可用性を確保する。EC2やELBはユーザーが自分で複数AZに展開する必要がある。「ユーザーが意識して展開すべきか」がポイント。
