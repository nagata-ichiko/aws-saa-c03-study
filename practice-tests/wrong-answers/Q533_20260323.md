# Q533 — Amazon SNS の通知エンドポイント

- **日付**: 2026-03-23
- **結果**: ❌ 不正解
- **自分の回答**: A, B (メール, CloudFront ディストリビューション)
- **正解**: A, D (メール, ショートメッセージサービス)
- **トピック**: Amazon SNS のサポートするエンドポイント
- **ポイント**: SNS がサポートするエンドポイントは Email, SMS, HTTP/HTTPS, SQS, Lambda, モバイルプッシュ, Kinesis Data Firehose。CloudFront, FTP, SNMP はサポートされていない。
