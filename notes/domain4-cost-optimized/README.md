### データ転送コストの最適化

AWS のデータ転送コストの基本原則：
- **インバウンド (AWS へのデータ転送)**: 基本的に無料
- **アウトバウンド (インターネットへのデータ転送)**: 有料
- **AZ 間のデータ転送**: 有料
- **リージョン間のデータ転送**: 有料

**コスト削減のポイント**:
- インターネットへの配信には **CloudFront** を使用する（EC2 から直接配信するより安価になることが多い）
- S3 や DynamoDB へのアクセスには **VPC エンドポイント (ゲートウェイ型)** を使用する（NAT ゲートウェイのデータ処理料金を回避できる）
