# サービス別学習ノート

各 AWS サービスの詳細ノートをここで管理します。
AI エージェントに「[サービス名] のノートを作成して」と指示することで、新しいサービスノートを追加できます。

## 📁 ファイル一覧

| ファイル名 | 対象サービス | 最終更新 |
| :--- | :--- | :--- |
| *(AI エージェントが学習を進めるにつれて追加されます)* | | |

## 🤖 AI エージェントへの指示例

- 「VPC のサービスノートを `notes/services/vpc.md` に作成して。SAA-C03 で重要なポイントを中心にまとめて」
- 「S3 のストレージクラスについて詳しくまとめて、`notes/services/s3.md` に追記して」
- 「IAM ポリシーの評価ロジックを図解を交えて解説し、`notes/services/iam.md` に記録して」
